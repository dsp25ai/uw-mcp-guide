Setup in 3 Steps

1

Get Your API Key

Go to Settings → API Dashboard and copy your API key.

2

Add the Config

Open your IDE's config file (see paths below). If the file doesn't exist, create it and paste the full config. If it already has content, add the "mcpServers" block as a new top-level key.

3

Restart Your Client

Fully quit and reopen your AI assistant (Claude Desktop, Cursor, VS Code, etc.) for the MCP server to connect. Just saving the file is not enough.

ⓘ
Important: The MCP server URL is https://api.unusualwhales.com/api/mcp. Do not use unusualwhales.com/public-api/mcp (this page) or mcp.unusualwhales.com (different server).

Already Have a Config File?

Add the "mcpServers" block as a sibling key next to your existing keys:

Before (your existing file)

{
  "preferences": {
    "coworkWebSearchEnabled": true,
    "sidebarMode": "chat"
  }
}

After (add mcpServers next to it)

{
  "preferences": {
    "coworkWebSearchEnabled": true,
    "sidebarMode": "chat"
  },
  "mcpServers": {
    "unusual-whales": {
      "command": "npx",
      "args": [
        "-y",
        "mcp-remote",
        "https://api.unusualwhales.com/api/mcp",
        "--header",
        "Authorization: Bearer YOUR_API_KEY"
      ]
    }
  }
}

IDE Configuration

No config files or terminal commands needed — set up entirely from the Claude Desktop UI.

1

Open Claude Desktop

Launch the Claude Desktop app on your computer.

2

Open Settings

Click the menu icon and select Settings.

3

Select "Connector"

Switch to the Connector tab in the settings panel.

4

Add Custom Connector

Click the "Add Custom Connector" button.

5

Enter Name & URL

NAME
Unusual Whales
URL
https://api.unusualwhales.com/api/mcp?token=YOUR_API_KEY
Replace YOUR_API_KEY with your actual key from Settings → API Dashboard

Connect through Perplexity's connector flow by pasting the JSON below.

1

Open Perplexity Settings

Go to the Connectors or MCP settings area.

2

Add a Custom Connector

Choose the option to add a custom MCP connector or server.

3

Paste the JSON

Replace <TOKEN> with your API key, then paste:

{
  "args": [
    "-y",
    "mcp-remote",
    "https://api.unusualwhales.com/api/mcp",
    "--header",
    "Authorization: Bearer YOUR_API_KEY"
  ],
  "command": "npx",
  "env": {},
  "useBuiltInNode": true
}
4

Save and Connect

Save the connector, then refresh or reopen Perplexity if the tools do not appear right away.

Troubleshooting

Make sure you are using the correct URL: https://api.unusualwhales.com/api/mcp. Common mistakes: using unusualwhales.com/public-api/mcp (that is the documentation page) or mcp.unusualwhales.com (a different, limited server). After updating your config, restart your AI assistant.
Verify your API key is correct and active. For remote config, ensure the Authorization header uses Bearer YOUR_API_KEY. For local config, check that UW_API_KEY is set in the env block.
Ensure Node.js 18+ is installed. Run node --version to check. If you use nvm, Claude Desktop may not find npx because it doesn't inherit your shell PATH. Fix this by using the full path to npx (e.g. /Users/you/.nvm/versions/node/v20.x.x/bin/npx) and add an env block with PATH.
Make sure the "mcpServers" block is a top-level key in your JSON file, at the same level as "preferences" or other existing keys — not nested inside them.
Fully quit and reopen your AI assistant after adding the MCP config — just saving the file is not enough. Check that the JSON is valid with no trailing commas.
The default rate limit is 120 requests/minute. Reduce concurrent queries or upgrade your plan for higher limits. The MCP server has built-in retry logic with exponential backoff.
Check your network connection. For remote config, verify you can reach api.unusualwhales.com. For local config, ensure npx can download the @unusualwhales/mcp package.
